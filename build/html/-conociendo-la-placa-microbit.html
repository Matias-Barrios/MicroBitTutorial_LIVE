MicroBit con Python

Una introducción básica a la programación en las placas micro:bit con Micro Python

🤖 Conociendo la placa Microbit

🤖 Conociendo la placa MicroBit

La tarjeta micro:bit concentra múltiples herramientas de hardware en un espacio muy reducido. Puedes identificar fácilmente cada uno de sus componentes clave usando el siguiente mapa visual:

  • Matriz LED 5x5: Ubicada al frente, funciona como una pantalla para mostrar letras, números o dibujos programados, y también actúa como sensor de luz ambiental.

  • Botones A y B: Dos pulsadores físicos situados a los lados de la pantalla que sirven para registrar la interacción del usuario y disparar eventos específicos en el código.

  • Pines de conexión (Edge Connector): La banda dorada inferior cuenta con agujeros grandes (0, 1, 2, 3V, GND) para conectar cables de cocodrilo o conectores tipo banana, expandiendo la placa hacia sensores y actuadores externos.

  • Procesador y Antena (Bluetooth/Radio): Ubicado en el reverso (Nordic nRF52833), es el cerebro que ejecuta las instrucciones y permite la comunicación inalámbrica entre tarjetas o con smartphones.

  • Sensores internos: Incluye un acelerómetro para detectar movimientos, inclinaciones o caídas, y una brújula digital (Compass) para reconocer la orientación magnética.
  • Puertos de alimentación y Reset: Cuenta con un conector Micro USB para programarla desde la computadora, un conector de batería JST para hacerla autónoma, y un botón de reinicio en la parte trasera.
