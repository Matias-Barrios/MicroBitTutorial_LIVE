MicroBit con Python

Una introducción básica a la programación en las placas micro:bit con Micro Python

🧠 ¿Que vamos a aprender en el curso?

🧠 ¿Qué vamos a aprender en el curso?

Este curso ofrece las herramientas esenciales para iniciarte en la programación y robótica educativa de forma rápida y entretenida a través de la acción directa. Para lograrlo, utilizaremos el simulador virtual de MakeCode de Microsoft, el cual replica con exactitud una tarjeta micro:bit real para que experimentes y pruebes tus códigos de forma inmediata sin necesidad de contar con el dispositivo físico.

Exploraremos a fondo la interfaz y el uso del emulador para gestionar proyectos e interactuar con los comandos básicos de la placa de forma intuitiva. Además, daremos un salto de calidad al aprender MicroPython, una versión optimizada de Python 3 que te permitirá escribir código real para controlar el hardware y interactuar con el entorno físico.

Finalmente, para que el proceso sea dinámico y motivador, vamos a ver algunos videos tutoriales divertidos. Estos recursos visuales están pensados para romper con la monotonía, mostrarte ejemplos de proyectos creativos ya terminados y guiarte de manera amena en la resolución de desafíos lógicos. Al combinar la potencia del emulador, la robustez del código y el apoyo de tutoriales entretenidos, terminarás el curso con la capacidad y la confianza necesarias para crear tus propias soluciones tecnológicas.
